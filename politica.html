Política de Privacidade

Última atualização: 03/2026

1. Responsável pelo Tratamento

O responsável pelo tratamento dos dados pessoais recolhidos através deste simulador é o titular do website.

2. Dados Recolhidos

Através do simulador podem ser recolhidos os seguintes dados:

  • Endereço de email
  • Número de telefone

3. Finalidade do Tratamento

Os dados recolhidos destinam-se exclusivamente a:

  • Contacto relacionado com a simulação efetuada
  • Prestação de informações adicionais solicitadas pelo utilizador

Os dados não serão utilizados para envio de comunicações não solicitadas.

4. Fundamento Legal

O tratamento dos dados baseia-se no consentimento explícito do utilizador, nos termos do Regulamento Geral sobre a Proteção de Dados (RGPD).

5. Conservação dos Dados

Os dados serão conservados apenas pelo período necessário para a finalidade para a qual foram recolhidos, ou até que o titular solicite a sua eliminação.

6. Partilha de Dados

Os dados não serão vendidos nem partilhados com terceiros, exceto quando exigido por obrigação legal.

7. Direitos do Titular dos Dados

Nos termos da legislação aplicável, o titular dos dados tem direito a:

  • Aceder aos seus dados
  • Retificar dados incorretos
  • Solicitar a eliminação dos dados
  • Retirar o consentimento a qualquer momento.

Para exercer qualquer um destes direitos, deverá contactar através do email disponibilizado no website.

8. Segurança

São adotadas medidas técnicas e organizativas adequadas para proteger os dados pessoais contra perda, uso indevido ou acesso não autorizado.

Este documento cumpre o Regulamento (UE) 2016/679 (RGPD).
